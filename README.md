## 🧠 About Me

I engineer **intelligent systems that protect networks**. My work lives at the intersection of machine learning and infrastructure security — specifically **DNS security, anomaly detection, and adversarial resilience**.

&gt; *"Research is only valuable when it ships."*  
&gt; I optimize for deployable, high-impact systems that survive real-world traffic.

- 🎓 Engineering Graduate, **ESPRIT Tunisia** (2025)
- 🔬 Research collaboration with **Landshut University (Germany)** & **Dräxlmaier**
- 🛡️ Core domains: DNS Security · AI Threat Detection · Distributed Protocols · Predictive Maintenance
- 🔭 Currently building next-gen DNS anomaly detection pipelines
- 🌱 Exploring LLM-based threat intelligence & adversarial ML
